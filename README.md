# suarezsport.github.io
Descripción del Proyecto
Suarez Sport Web es una landing page promocional y de catálogo para una marca de ropa deportiva que se especializa en la personalización y estampados exclusivos. El sitio está diseñado para informar a los clientes sobre la gama de productos disponibles (camisas, pantalones, conjuntos, suéteres) y resaltar las ventajas de elegir una marca enfocada en la calidad y el diseño a la medida.

El objetivo principal de la web es facilitar el contacto para pedidos personalizados y resolver dudas comunes.

✨ Contenido y Secciones Clave
La página guía al usuario a través de la propuesta de valor de la marca:

Banner Principal: Presenta la promesa de la marca: "Ropa deportiva personalizada a tu estilo".

Ventajas de Elegir Suarez Sport: Destaca los pilares fundamentales del negocio:

Personalización Fácil: Opciones de personalización como logos y colores.

Calidad Superior: Uso de materiales de alta calidad para durabilidad y confort.

Variedad de Productos: Amplio catálogo para todas las necesidades deportivas.

Atención al Cliente: Soporte para elegir y personalizar la ropa.

Testimonio: Una prueba social para generar confianza en la calidad y el servicio.

Preguntas Frecuentes (FAQ): Responde a las dudas clave sobre:

Tipos de personalización (estampados, bordados).

Tiempos de entrega (7 a 14 días hábiles).

Política de devoluciones/cambios en productos personalizados.

Descuentos por pedidos al por mayor.

Contacto: Sección para que los clientes envíen mensajes directamente, además de la ubicación física (Paraguachí, Nueva Esparta).

Redes Sociales: Enlaces directos a Instagram, Facebook y WhatsApp para una comunicación rápida y efectiva.

🛠️ Tecnologías Utilizadas
El sitio está construido con una base sencilla para asegurar rapidez y eficiencia:

HTML: Estructura de la tienda en línea.

CSS: Estilos y presentación visual.

Alojamiento: Vercel.
